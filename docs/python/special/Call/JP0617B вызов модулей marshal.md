# JP0617B
## Вызов модулей marshal
Использование marshal может быть небезопасным при десериализации данных из недоверенного источника.
Данный тест предупреждает об использовании marshal для десериализации данных.

---
Список вызовов:

* marshal.loads
* marshal.load
