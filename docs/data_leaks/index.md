# Утечка данных

Помимо статистического анализатора кода также умеем искать разного рода утечки данных:

- пароли
- токены
- api-ключи
- безопасные оболчки
- ...

Пример работы можно найти в конце на [главной странице](https://docs.jethub.pro/#_3).
